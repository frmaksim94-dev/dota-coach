Dota Coach AI 0.9.1 — что исправлено
=====================================

1. Таблицы
----------
Исправлены белые полосы в таблицах. Теперь строки таблиц темно-синие, текст светлый и должен нормально читаться на всех строках.

2. Иконка приложения в панели задач Windows
-------------------------------------------
Добавлена явная Windows AppUserModelID-настройка. Это помогает Windows показывать иконку Dota Coach AI в панели задач вместо обычной иконки Python/pythonw.

Важно: если запускаешь проект через DotaCoach.vbs или DotaCoach.pyw, Windows иногда всё равно может группировать окно как Python. Самый надежный вариант для красивой иконки в панели задач — собрать exe через build_windows_exe.bat и запускать dist\Dota Coach AI\Dota Coach AI.exe.

3. Как включить Live Mode
-------------------------
Live Mode работает через Dota 2 Game State Integration. Приложение не читает память игры и не нажимает кнопки. Dota сама отправляет разрешенные live-данные локальному серверу приложения.

Порядок:

1) Открой Dota Coach AI.
2) Перейди во вкладку Live Coach.
3) Нажми "Запустить Live GSI".
4) Нажми "Установить cfg в Dota 2".
5) Если автоустановка не нашла Dota 2, нажми "Сохранить cfg".
6) Скопируй файл gamestate_integration_dota_coach.cfg в папку Dota 2:

   steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\

   Если папки gamestate_integration нет, создай ее вручную.

7) В Steam открой:

   Библиотека -> Dota 2 -> Свойства -> Общие -> Параметры запуска

8) Добавь туда:

   -gamestateintegration

9) Полностью закрой и заново запусти Dota 2.
10) Зайди в матч или лобби/бот-матч.
11) В приложении на вкладке Live Coach статус должен смениться с "данных пока нет" на "Live: последний snapshot ... сек. назад".

Если Live Mode не заработал
---------------------------
- Проверь, что файл лежит именно в:
  steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\

- Проверь, что имя файла начинается с:
  gamestate_integration_

  В проекте имя уже правильное:
  gamestate_integration_dota_coach.cfg

- Проверь, что в параметрах запуска Dota 2 есть:
  -gamestateintegration

- Перезапусти Dota 2 после копирования cfg.

- Сначала нажми "Запустить Live GSI" в приложении, потом запускай матч.

- Если Windows спросит разрешение для сети/брандмауэра, разреши для локальной сети. Сервер слушает только локальный адрес 127.0.0.1:3000.

- Если порт 3000 занят другой программой, поменяй DOTA_COACH_GSI_PORT в переменных окружения или закрой программу, которая заняла порт.
