Dota Coach AI 0.9.6

Что изменено:

1. Pro Lab карты
- Старые абстрактные схемы заменены на более похожие на minimap Dota: Radiant/Dire зоны, река, линии, лес, вышки, кемпы, Рошан и маршруты.
- Добавлена кнопка "Обновить карту Dota". Если есть интернет, приложение попробует скачать чистую карту и перерисовать маршруты поверх неё.
- Если источник карты недоступен, остаются встроенные улучшенные схемы.

2. Иконки героев и предметов
- Кнопка "Обновить базу и иконки" теперь подтягивает реальные портреты героев и иконки предметов через OpenDota/Steam CDN.
- Иконки сохраняются не внутрь exe, а в пользовательский кеш:
  %LOCALAPPDATA%\DotaCoachAI\assets_cache\
- Если интернета нет, приложение открывается с резервными иконками.

3. Аккаунты разных игроков
- В верхнее поле можно вставить account_id, Steam64, ссылку OpenDota /players/... или Steam profile URL /profiles/...
- Профили можно сохранить и переключать в выпадающем списке.
- Для другого человека достаточно отправить ему папку приложения и инструкцию: он вводит свой account_id/Steam64 и нажимает "Обновить".

4. Оформление
- Тема стала менее чёрно-белой: добавлены синие, оранжевые и градиентные акценты, более заметные скроллбары и карточки.

5. История и мета
- История матчей работает для выбранного аккаунта, если профиль матчей публичный.
- Вкладка "Лучшие герои" обновляет винрейты и пики через OpenDota heroStats.

Важно:
- DotaBuff не имеет удобного публичного API для такого приложения, поэтому автоматическое обновление сделано через OpenDota API и Steam CDN для изображений.
- Если профиль Steam/Dota закрыт, OpenDota может не отдать историю матчей.
- Ollama нужна только для расширенных LLM-ответов во вкладке "ИИ Тренер". Базовая аналитика работает без неё.
