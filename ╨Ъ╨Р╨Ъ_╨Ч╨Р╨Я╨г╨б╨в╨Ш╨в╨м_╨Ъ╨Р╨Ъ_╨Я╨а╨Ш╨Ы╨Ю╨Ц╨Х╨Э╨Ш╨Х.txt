Dota Coach AI — запуск как обычного приложения на Windows
==========================================================

Вариант 1 — самый простой, без сборки EXE
----------------------------------------
1. Распакуй архив в удобную папку, например: C:\Games\DotaCoach
2. Один раз запусти файл setup_windows.bat
3. После установки запускай приложение через ярлык "Dota Coach AI" на рабочем столе
   или двойным кликом по файлу DotaCoach.vbs.

Консоль при обычном запуске открываться не будет.

Вариант 2 — собрать настоящий EXE
---------------------------------
1. Распакуй архив.
2. Запусти build_windows_exe.bat
3. Готовое приложение появится здесь:
   dist\Dota Coach AI\Dota Coach AI.exe

Этот EXE запускается как обычная программа, без консоли.

Если приложение не запускается
------------------------------
- Проверь, что установлен Python 3.11 или новее.
- При установке Python включи галочку "Add python.exe to PATH".
- Если запуск через DotaCoach.vbs не сработал, посмотри файл startup_error.log
  в папке проекта.

Иконка в панели задач
---------------------
Приложение задаёт Windows AppUserModelID. Если при запуске через VBS Windows всё равно показывает Python-иконку, удали старый ярлык, заново запусти setup_windows.bat или собери EXE через build_windows_exe.bat.
