# Запуск Dota Coach AI как обычного Windows-приложения

## Что исправлено в версии 0.9.2

- В таблицах убраны светлые полосы: строки принудительно темные, текст светлый.
- Для Windows оставлен AppUserModelID, чтобы панель задач показывала иконку Dota Coach AI, а не Python.
- GSI-сервер стартует автоматически вместе с приложением.
- Во вкладке **Live Coach** добавлены **Тест сервера** и **Диагностика cfg**.
- Автоустановка GSI cfg теперь пишет файл и напрямую в `game\dota\cfg`, и в `game\dota\cfg\gamestate_integration`.
- Добавлена вкладка **Pro Lab** с картинками-паттернами, свежими pro-матчами и replay/demo загрузкой.

## Вариант 1 — двойной клик без сборки `.exe`

1. Распакуй архив в удобную папку, например:

```text
C:\Games\DotaCoach
```

2. Один раз запусти:

```text
setup_windows.bat
```

Этот файл создаст виртуальное окружение `.venv`, установит зависимости и создаст ярлык **Dota Coach AI** на рабочем столе.

3. После установки запускай приложение через ярлык на рабочем столе или двойным кликом по файлу:

```text
DotaCoach.vbs
```

`DotaCoach.vbs` запускает `DotaCoach.pyw` через `pythonw.exe`, поэтому обычная консоль при запуске не открывается.

## Вариант 2 — собрать настоящий `.exe`

1. Запусти:

```text
build_windows_exe.bat
```

2. После сборки готовая программа появится здесь:

```text
dist\Dota Coach AI\Dota Coach AI.exe
```

Этот файл можно перенести на рабочий стол или закрепить в меню «Пуск». Он собирается в режиме `console=False`, поэтому запускается без консольного окна.

## Исправление иконки в панели задач

Приложение задаёт Windows AppUserModelID, поэтому окно должно получать иконку **Dota Coach AI** в панели задач. Если Windows всё равно показывает иконку Python, удали старый ярлык, запусти `setup_windows.bat` заново или собери настоящий `.exe` через `build_windows_exe.bat` — для панели задач это самый надёжный вариант.

## Где сохраняется GSI cfg

Во вкладке **Live Coach** нажми **Установить cfg в Dota 2**. Приложение попробует записать файл в несколько мест, включая главный путь:

```text
steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration_dota_coach.cfg
```

Если автоустановка не нашла папку игры, нажми **Сохранить cfg**. Когда приложение собрано в `.exe`, сгенерированный cfg сохраняется в пользовательскую папку:

```text
%LOCALAPPDATA%\DotaCoachAI\config\gamestate_integration_dota_coach.cfg
```

Скопируй его вручную прямо в:

```text
steamapps\common\dota 2 beta\game\dota\cfg\
```

Дополнительно можно скопировать и сюда:

```text
steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\
```

## Если Live Coach всё равно пишет «данных пока нет»

1. Во вкладке **Live Coach** нажми **Тест сервера**.
2. Если тест прошел — приложение работает, проблема в cfg/запуске Dota.
3. Нажми **Диагностика cfg** и проверь, видит ли приложение установленный файл.
4. Проверь параметры запуска Dota 2:

```text
-gamestateintegration
```

5. Полностью перезапусти Dota 2 после установки cfg.
6. Убедись, что внутри cfg стоит:

```text
"uri" "http://127.0.0.1:3000/"
```

## Если приложение не запускается

- Установи Python 3.11 или новее.
- При установке Python включи галочку `Add python.exe to PATH`.
- Запусти `setup_windows.bat` заново.
- Если ошибка появляется после запуска без консоли, посмотри файл `startup_error.log` в папке проекта.
- Если собираешь `.exe`, нужен интернет для установки PyInstaller и зависимостей.
